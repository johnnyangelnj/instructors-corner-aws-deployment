We recommend you download the application and practice delivering a course presentation a few times so you are familiar with the technology before using it in class. Although printed reference materials are not necessary when using the course presentation, it is recommended that you have back-up copies of the ready reference cards, activity resources and DVD in case technical difficulties occur.
