Single Student Certification per Course Taken – Beginning Feb. 1, 2013

In response to your suggestions for reducing the number of certificates issued for courses with multiple components, we have streamlined this process. Beginning February 1, 2013, we will discontinue issuing certifications for multiple components in a course, and will issue a single certificate for the course taken that reflects all components in the course. This should make it simpler for individuals and organizations to keep track of certification records and validity periods for the affected courses.

In implementing this change, instructors have to be mindful of two important changes, outlined below:
(1) the use of new Course Code Versions
(2) the need to separate course records if not all students take all the same components during a class



New Course Code Versions

  • The single certification per course taken applies only to courses that have a course code starting with the following letters: HSSSFA, HSSPRO, HSSRTE and HSSBBP.

  • The current versions of all HSSRTE, HSSPRO, HSSSFA, and HSSBBP courses will be discontinued on 1/31/2013. New course versions must be used for training in these categories set up to take place on or after February 1, 2013.

    Course is held on:

    Use Course Version:

    Now – 1/31/2013

    r.2011, r.2012, or blank

    2/1/2013 – forward

    r.2012(n)

  • Please note that course records submitted using the wrong course version based on the date of the course will be rejected by the Training Support Center, and will need to be resubmitted.

  • The new single certificate name will typically match the course title.

  • Exceptions to the single certification will include courses with components that have varying validity periods (i.e. a combination course of RTE and Adult First Aid/CPR/AED and BBP) and/or when the name of the course exceeds the 100 characters allowable per printed certification.

  • The single certification process will not apply to Aquatics or Caregiving courses such as Lifeguarding, Babysitting, and others.

  • Detailed directions for this activity can be found in the Creating a Course Record as a CR Administrator Job Aide found at instructorscorner.org/files/843.

 

NOTE: Instructors must pay attention to selecting the appropriate course version code for classes being set up between now and February 1, 2013 to avoid a processing error. The new version codes are live in the Learning Center and must be used for training scheduled to take place on or after February 1, 2013. Course records for training that will/has take(n) place on or before 1/31/2013 must continue to use the current course version codes.

For example: For an Adult First Aid/CPR/AED course (HSSSFA404):

  • If setting up the class between now and February 1, with the class taking place BEFORE February 1:

    • Use the r.2011 or r.2012 version of the course

    • Students will receive three certificates, one for each component – Adult CPR, Adult AED, and First Aid.

     

  • If setting up the class now or going forward, with the class taking place ON or AFTER February 1:

    • Use the ‘r.2012(n) version of the course

    • Students will receive one certificate titled Adult First Aid/CPR/AED. 

     
 


Classes With Students Taking Different Components

If all students in a training class take the same components, there is no special action that must be taken, and the instructor will submit a single Course Record for the class. However, there is no longer an option to combine students who participate in portions of a class with students who complete all parts of the class on the same course record. 

  • When submitting course records in the Learning Center, continue to create course records based on the course codes that represent the correct components each student participated in. Payment will need to be entered separately for each course record if you are using the Learning Center. 

  • When submitting course records via email or postal mail, you will need to submit a separate course record for each variation in student participation.

    For example: Instructor teaches an Adult First Aid/CPR/AED and there are 5 students. Billy and Jane take the entire course and pass. Suzy and Harriet only participate in the First Aid portion and pass. Joe only needs Adult CPR. Instructor must submit three course records [(i) for Adult First Aid/CPR/AED (HSSSFA404), (ii) for Adult CPR (HSSSFA102) and (iii) for First Aid (HSSSFA101)] to ensure that each student gets only the certificate(s) they earned. 

    •     

  • Separating students into multiple course records based on the actual components they completed will also ensure that you are charged appropriately for each course. 

 

* * *

We are pleased to begin offering these new streamlined certifications to you and the students you train. If you have questions regarding these changes, please contact us at 1-800-REDCROSS (1-800-733-2767) or support@redcrosstraining.org. As always, thank you for your continued support of the American Red Cross and your direct efforts in making our communities safer.

Clicking the ‘picker icon’ will bring up the “Select Course” window shown below. If you do not see a “Version” column, select the Modify Table link and choose to display that column.
