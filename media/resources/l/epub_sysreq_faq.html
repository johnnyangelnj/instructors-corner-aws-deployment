ePUB System Requirements & FAQ


ePUB is a standard e-book file format that is supported by most e-readers available today. The ePUB format is popular because it allows a book's text to reflow automatically to different screen sizes, including different e-readers. You can also read ePUB books on a desktop or mobile phone by using e-reader software, many of which can be downloaded for free:

Stanza
Mobipocket
FBReader (for Linux)
Aldiko (for Android)
Adobe Digital Editions
Microsoft Reader


Which devices are compatible with the ePUB file format?
The ePUB file format is an open e-book file format that is the industry standard. With the exception of the Amazon Kindle (which uses its own proprietary software), most devices can download and read ePUB files.


Do I need to be connected to the Internet to read my ePUB book?
No. Once you've downloaded the ePUB file to your device, it is accessible to you anytime.


Can I share my ePUB book with others?
No. The file is licensed for your personal reading only, so you are not legally allowed to copy, share, or email the book to others. Likewise, you are not allowed to resell or distribute the book to others.
